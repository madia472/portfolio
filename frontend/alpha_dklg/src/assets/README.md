# Assets Directory

Ce dossier contient tous les fichiers statiques du portfolio.

## Structure des dossiers

```
assets/
├── images/          # Images du portfolio
│   ├── profile/     # Photos de profil
│   └── projects/    # Images des projets
├── documents/       # Documents (CV, certificats, etc.)
└── icons/          # Icônes personnalisées
```

## Formats acceptés

### Images
- Format recommandé : JPG, PNG, WebP
- Taille maximale : 2MB
- Résolution recommandée pour les photos de profil : 500x500px

### Documents
- Format recommandé : PDF
- Taille maximale : 5MB

## Nomenclature des fichiers

- Utilisez des noms en minuscules
- Séparez les mots par des tirets (-)
- Évitez les caractères spéciaux et les accents
- Incluez la date pour les CV (exemple : cv-mamadou-diallo-2024.pdf)

Exemple :
```
photo-profil.jpg
cv-mamadou-diallo-2024.pdf
projet-portfolio.png
```
